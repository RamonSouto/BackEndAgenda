# 🏥 Sistema de Agenda de Ajustes Pós-Cirúrgicos

Sistema completo para gerenciamento de agenda de ajustes de produtos pós-cirúrgicos, desenvolvido com Node.js, Express e MySQL.

## 📋 Características

- ✅ Autenticação JWT com controle de permissões
- ✅ Cadastro de pessoas (Administrador, Secretária, Costureira, Paciente, Médico)
- ✅ Gestão de agendamentos com regras de negócio
- ✅ Upload de fotos de ajustes
- ✅ Sistema de logs completo para auditoria
- ✅ Notificações via WhatsApp
- ✅ Relatórios e dashboard
- ✅ Arquitetura baseada em princípios SOLID
- ✅ Segurança com Helmet, Rate Limiting e sanitização

## 🚀 Tecnologias

- **Node.js** v16+
- **Express** v4
- **MySQL** v8
- **JWT** para autenticação
- **Bcrypt** para hash de senhas
- **Multer** para upload de arquivos
- **Joi** para validação de dados
- **Date-fns** para manipulação de datas
